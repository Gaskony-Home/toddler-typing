# Roadmap

Future plans for Toddler Typing.

## Current Features (v1.x)

- Letters & Numbers learning
- Colors & Shapes exploration
- Drawing canvas
- Coloring pages
- Dot-to-dot games
- Animal sounds
- Animated dinosaur character
- Windows desktop + Android tablet
- Keyboard lock (Windows)
- Fullscreen mode

## Planned Features

### Phase 1: Polish & Sound
- [ ] Letter/number pronunciation (TTS)
- [ ] Success sound effects
- [ ] Background music (optional)
- [ ] Animated celebrations (confetti, stars)

### Phase 2: More Activities
- [ ] Counting with visual objects
- [ ] Matching games (shapes, colors, letters)
- [ ] Pattern recognition
- [ ] Memory card game
- [ ] Simple puzzles

### Phase 3: Progress & Gamification
- [ ] Star rewards for correct answers
- [ ] Level progression system
- [ ] Achievement badges
- [ ] Progress tracking for parents
- [ ] Multiple child profiles

### Phase 4: Accessibility & Languages
- [ ] Text-to-speech for all content
- [ ] High contrast mode
- [ ] Multiple language support
- [ ] Colorblind-friendly palettes

## Contributing

Ideas welcome! Open an issue on GitHub with the `enhancement` label.

## Core Values

- **Safety First**: No ads, no purchases, no external links
- **Privacy**: No data collection, completely offline
- **Child-Focused**: Age-appropriate, encouraging, fun
- **Open Source**: Transparent, auditable code
