# Assets Directory

This directory contains assets for the Toddler Typing application.

## Structure

- `images/` - Image files (icons, backgrounds, educational graphics)
- `sounds/` - Sound effects and audio files
- `fonts/` - Custom fonts for the application

## Adding Assets

When adding assets:

1. Place them in the appropriate subdirectory
2. Use descriptive filenames (e.g., `letter_a_sound.wav`, `success_sound.wav`)
3. Keep file sizes reasonable for better performance
4. Supported formats:
   - Images: PNG, JPG, BMP
   - Sounds: WAV, OGG, MP3
   - Fonts: TTF, OTF
